# LOCALS
Contains Dataset, Implementation and Results for LOCALS.

# Tensorflow Model Versions
You can find the [models](https://www.kaggle.com/models/agilesharumugam/locals) here.
  1. LOCALS (Paper Version) -> Tensorflow 2.18.0
  2. LOCALS2 (New Version) -> Tensorflow 2.10.0
  3. LOCALSLightweight (Lightweight Version) -> Tensorflow 2.10.0

# Dataset
You can find the [dataset](https://www.kaggle.com/datasets/agilesharumugam/locals-dataset-paper-and-new-version) here.
  1. Paper Version -> Used for the making of the paper version of model.
  2. New Version -> Used for the making of both the new version and lightweight version.
